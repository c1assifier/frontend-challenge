# VK Cat Pinterest

## Описание

**VK Cat Pinterest** — это SPA-приложение для просмотра котиков, разработанное в рамках тестового задания.
В приложении реализованы вкладки со всеми котиками и избранными, хранение избранного на клиенте и бесконечная подгрузка карточек.

---

## Демо

[Посмотреть live demo](https://c1assifier.github.io/frontend-challenge/)

## Используемые технологии

- React
- TypeScript
- Vite
- React Icons
- ESLint
- Prettier

---

## Функциональность

- Загрузка котиков с TheCatAPI
- Вкладка "Все котики"
- Вкладка "Любимые котики"
- Добавление и удаление котиков из избранного
- Хранение избранного в `localStorage`
- Бесконечный скролл с подгрузкой новых карточек
- Lazy loading изображений

## Запуск проекта

1. Клонировать репозиторий

```bash
git clone <ссылка-на-репозиторий>
```

2. Перейти в директорию проекта

```bash
cd frontend-challenge
```

3. Установить зависимости

```bash
npm install
```

4. Запустить проект

```bash
npm run dev
```
