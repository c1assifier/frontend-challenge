# Cat Pinterest

Тестовое задание для стажировки Front-end разработчика в Учи.ру.

## Описание

Приложение для просмотра изображений котиков с возможностью добавления в избранное.

Данные загружаются с публичного API:

- https://thecatapi.com

Интерфейс реализован на основе предоставленного макета:

- https://bit.ly/3utxaL2

---

## Функциональность

- Просмотр всех котиков
- Добавление/удаление из избранного
- Отображение избранных котиков
- Хранение данных на клиенте (localStorage)

---

## Технологии

- React
- TypeScript
- Vite
